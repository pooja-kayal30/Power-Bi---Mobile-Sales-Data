📌 Title: Mobile Sales Data Analysis Dashboard (Power BI)

📄 Description:

This project presents an interactive Power BI dashboard built using a Mobile Sales dataset to analyze sales performance, customer trends, and product insights.

The dashboard provides a clear understanding of business performance through KPIs, charts, and filters, enabling better decision-making.

🔷 📊 Project Overview
Analyzed mobile sales data using Power BI
Created interactive dashboards with filters and slicers
Visualized key metrics like Sales, Profit, Quantity, and Orders
Focused on trends by City, Time, and Product Category
🔷 ⚙️ Steps Followed (Step-by-Step)
1️⃣ Data Collection
Imported Mobile Sales dataset into Power BI
Dataset includes:
Order Date
Sales
Quantity
City
Product Details
2️⃣ Data Cleaning & Transformation
Removed unnecessary columns
Handled missing values
Changed data types (Date, Number, Text)
Created proper relationships (if multiple tables)
3️⃣ Data Modeling
Built relationships between tables
Organized data for better performance
Created calculated columns like:
Day Name
Delivery Days
4️⃣ DAX Calculations

Created measures for analysis:

Total Sales
Total Profit
Total Quantity
Total Orders
Last Year Sales (for comparison)
5️⃣ Dashboard Creation

Designed an interactive dashboard with:

KPI Cards (Sales, Profit, Orders, Quantity)
Bar Charts (Sales by City/Product)
Line Chart (Sales Trend Over Time)
Map Visual (City-wise Sales)
Slicers (Date, Category, Region)
6️⃣ Design & UI
Used modern dark/light theme
Applied proper colors and formatting
Added titles and labels for clarity
Created a user-friendly layout
7️⃣ Insights Generated
Identified top-performing cities
Analyzed sales trends over time
Found high-demand products
Compared current vs previous performance
🔷 📈 Key Features

✔ Interactive dashboard
✔ Clean and structured data model
✔ DAX-based calculations
✔ Business insights generation

🔷 📂 Files Included
.pbix file (Power BI Dashboard)
Dataset (Excel/CSV)
🔷 🎯 Conclusion

This project demonstrates how Power BI can be used to transform raw data into meaningful insights and visually appealing dashboards for business analysis.
